# Linux Fundamentals Practice Project

This project demonstrates basic Linux command-line skills practiced using macOS Terminal.

## Skills practiced
- Directory navigation (cd, ls)
- File creation (echo, mkdir)
- Output redirection (>, >>)
- File searching (find)
- Text searching (grep)
- Basic log analysis
